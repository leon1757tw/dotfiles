## Installation

**Requires [GNU stow](https://www.gnu.org/software/stow/).**

To install a specific package of dotfiles (take i3wm for example)
```shell
$ stow i3
```

To install everything
```shell
$ # This script temporarily hides README.md, and execute `stow *`
$ ./.install.sh
```
